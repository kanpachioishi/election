# Docs Index

`docs/` は内部運用文書の置き場です。

## Subdirectories

- `audits/`: coverage checks and source coverage status
- `inventories/`: source inventories and dataset inventories
- `notes/`: small editorial notes, trivia, and reusable story ideas
- `plans/`: roadmaps, rollout plans, and expansion plans
- `policies/`: editorial and verification policies
- `runbooks/`: operating procedures and deploy workflows
- `specs/`: product, data, and feature specifications
- `templates/`: reusable writing or page templates
