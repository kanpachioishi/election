# 現職市長一覧 小ネタノート

最終更新: 2026-04-24

## 同姓候補が出ると得票が小数点になることがある

南丹市長選挙（2026-03-15執行）では、候補者が

- `西村 よしたか`
- `西村 良平`

の2人で、どちらも姓が `西村` だった。

この場合、投票用紙に `西村` とだけ書かれた票は、どちらの候補への投票か特定できない。  
そのため、公職選挙法上の `按分` により、得票割合に応じて票が分けられる。

結果として、開票結果は小数点付きになった。

- `西村 よしたか`: `8,547.496票`
- `西村 良平`: `6,424.503票`
- `按分切捨`: `0.001票`

合計は `14,972.000票` になり、有効投票数と一致する。

## サイトでの使いどころ

- 「選挙結果の数字はなぜ小数点になるのか」という導入に使える
- 「名字だけ書くとどう扱われるか」という投票の仕組み紹介に使える
- 市長一覧や選挙解説ページのコラム欄に再利用できる
